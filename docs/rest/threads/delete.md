# Delete Thread

### Resource URL

`https://api.multividas.com/rest/v1/threads/{thread:id}`

### Resource Information

| Method                                | DELETE  |
|---------------------------------------|---------|
| Response formats                      | JSON    |
| Requires authentication?              | YES     |

## Response [200]

```json
{
    "info": "Thread deleted",
    "code": 200,
    "meta": {}
}
```
