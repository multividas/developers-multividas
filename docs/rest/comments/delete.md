# Delete Comment

### Resource URL

`https://api.multividas.com/rest/v1/comments/{comment:id}`

### Resource Information

| Method                                | DELETE  |
|---------------------------------------|---------|
| Response formats                      | JSON    |
| Requires authentication?              | YES     |

## Response [200]

```json
{
    "info": "Comment deleted",
    "code": 200,
    "meta": {}
}
```
